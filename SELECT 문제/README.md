# 서울에 위치한 식당 정보 조회 SQL

## 📌 요구사항
- `REST_INFO`와 `REST_REVIEW` 테이블에서 **서울에 위치한 식당**을 조회
- 조회할 컬럼:
    - **식당 ID**
    - **식당 이름**
    - **음식 종류**
    - **즐겨찾기 수**
    - **주소**
    - **리뷰 평균 점수 (소수점 3번째 자리에서 반올림)**
- **정렬 기준:**
    1. **리뷰 평균 점수**(내림차순)
    2. **평균 점수가 같다면 즐겨찾기 수**(내림차순)
